How It Works

A daily agent process reads recent changes from all registered projects and produces targeted intelligence briefs. Each brief summarizes what one project discovered that's relevant to another. The process is defined in cross-pollination/process/daily-sweep.md and the project registry lives in cross-pollination/projects.json.
