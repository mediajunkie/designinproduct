# Cross-Pollination Briefs

Daily intelligence briefs synthesizing insights across sibling projects.

## Format

Each brief is a Markdown file named `YYYY-MM-DD-brief.md` with YAML frontmatter:

```yaml
---
date: 2026-03-21
status: substantive | nominal | skip
sources_checked:
  - klatch
  - piper-morgan
---
```

**Status values:**
- `substantive` — Cross-relevant insights found; full brief
- `nominal` — Sweep completed, nothing noteworthy; brief confirms the sweep ran
- `skip` — No brief generated (system offline, holiday, etc.)

## Reading a Brief

Briefs are organized by insight, not by project. Each Key Insight includes:
- Which project it came from
- Why it matters to the other project(s)
- A link to the source file
- A suggested action

## For Agents

If you're an agent on Klatch or Piper Morgan and want to check for cross-project updates, read the most recent brief in this directory. Your project's daily sweep or session start protocol can include checking this location.
